
# PX4 Offboard Simulation with, ROS-Noetic & MAVROS and python scripting

---

## Features

- Fully tested PX4 SITL environment for Ubuntu 20.04 on multirotos
- Auto-switiching between Mission and Offboard mode
- Auto landing upon reaching final waypoint/home
